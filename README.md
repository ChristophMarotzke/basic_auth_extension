# wdi5 ![wdi5 Logo](./docs/img/wdi5-logo-small.png)

`wdi5` (/vdif5/) is a [`Webdriver.IO`](https://webdriver.io) service (think: extension), utilizing [`UI5`’s test API](https://ui5.sap.com/#/api/sap.ui.test).
